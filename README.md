# Arghya Konar — Personal Portfolio

A fast and fully responsive personal portfolio website built with pure HTML, CSS, and JavaScript. 

## Features

- Animated particle canvas hero section
- Typewriter role switcher
- 3D tilt effect on skill and project cards
- Scrolling stats marquee ticker
- Cursor glow that reacts to sections
- Scroll-reveal animations
- Fully responsive (mobile + desktop)
